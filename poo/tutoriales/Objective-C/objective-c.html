Tutorial Objective C

Objective C es un lenguaje orientado a objetos creado como un superconjunto de C. Fue creado por Brad Cox y la corporación StepStone en 1980

Actualmente se usa como un lenguaje principal de programación para Mac OS X, iOS y GNUstep
